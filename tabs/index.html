There were very few issues, the main one being where focus is placed after navigating through the page; focus takes the user to the address bar and not the main heading as expected. Users would have to tab through the page again to restart navigating the tab buttons.

If the reading order of the page continues logically after the content panel then there will be no need for any adjustments to be made. [This has since been implemented and thus resolves any outstanding issues - Justin Perry 02/06/2014]
